# Language Practice Space

A standalone local-first language practice app extracted out of Astroverse.

Astroverse should remain astrology-only. This repository is the proper home for language learning UI, phrase packs, pronunciation notes, saved local practice answers, and future language-coach work.

## Current scope

- Spanish, Mandarin, and Korean starter phrase packs
- Literal and natural translations
- Pronunciation/romanization notes
- Local-only saved answers using browser localStorage
- No backend required

## Run locally

Open `index.html` directly, or run:

```bash
python3 -m http.server 8088
```

Then visit:

```text
http://127.0.0.1:8088
```
